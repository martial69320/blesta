# WHMSonic Module

A Blesta module that integrates with [WHMSonic](https://www.whmsonic.com).

Originally created by Phillips Data, Inc.

**To use the module, move it to /components/modules/ and rename it whmsonic.**

### Blesta Compatibility

|--------------|--------------|
|< v4.2.0|v1.0.1|
|>= v4.2.0|v1.1.0|
